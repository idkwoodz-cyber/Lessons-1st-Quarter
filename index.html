  • XHTML

    An example of an XHTML rule is the use of proper nesting. For example: " This text appears in Boldface and Itallic "

  • HTML Lists

    HTML Lists are used to organize information in a code. The 2 main types of lists are Unordered and Ordered lists. Here is an example:

    • BLACKPINK
    • TWICE
    • 2NE1

    1. BTS
    2. SEVENTEEN
    3. ENHYPEN

    You may also nest lists within lists.

  • HTML Tables

    HTML Tables are also used to organize information in rows and columns. You can code many things into a table, like pictures, text, and even lists! You can also merge the table cells to make it look more organized and to your liking.

    Santi's Table
    Item Type Likes Dislikes
    I. Foods Mangoes, Oranges, Grapes Durian, Rambutan, Apples
    II. Subjects English, MAPEH, TLE/ICT None!
    III. Animals Bunnies, Cats, Dogs Snakes, Bugs, Coyotes

    This is an example of the table I used in my Home Page. By far, tables are the longest code I've typed yet. Similar to lists, you may also nest tables within tables.

  • HTML Hyperlinks

    Hyperlinks in an HTML code are made to redirect you to another website or domain. It's useful if you have a lot of pages on your website and you need an organized redirection. Hyperlinks are able to redirect you to almost anything, a published website, an e-mail, etc. Here are some examples:

    My Favorite Website
    Message Me!

    Hyperlink No. 1 redirects you to my favorite website, while Hyperlink No. 2 redirects you to an e-mail draft to the recipient typed on the code.

  • HTML Forms

They can input text, select options, and submit the user input. Example of a form with a text box, radio buttons, check boxes, drop-down list, and submit & clear buttons:
